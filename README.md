# Sword In The Stone

A fantasy-themed Blender environment featuring a glowing sword embedded in stone inside a cave.

## Tools Used

- Blender
- Cycles Renderer

## Features

- Custom sword model
- Emissive blade effect
- Atmospheric cave lighting
- Stone environment
- Cinematic composition
